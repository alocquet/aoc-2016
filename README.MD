<h1>Soluces - Advent of code 2016</h1>
Soluces for Advent of code 2016 written with TypeScript and mocha tests.

<h2>Build</h2>
* clone git
```shell
git clone https://github.com/alocquet/aoc-2016.git
```
* install
```shell
npm i
```
* run tests
```shell
npm test
```
